# ⚖️ Tawazon – AI Nutrition & Wellness Coach

> Your intelligent nutrition companion for healthier living.

## Overview

Tawazon is an AI-powered mobile application designed to help users improve their health through personalized nutrition guidance, meal analysis, healthy habit tracking, and intelligent coaching.

The application combines modern artificial intelligence with evidence-based nutritional principles to deliver practical daily recommendations while maintaining a simple and user-friendly experience.

# Installation

Clone the repository

```bash
git clone https://github.com/your-username/tawazon.git
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm start
```

Run Android

```bash
npm run android
```

Run iOS

```bash
npm run ios
```

---

# Environment Variables

Create a `.env` file and configure:

```
OPENAI_API_KEY=

DATABASE_URL=

FIREBASE_API_KEY=

FIREBASE_PROJECT_ID=

REVENUECAT_API_KEY=

SUPABASE_URL=

SUPABASE_ANON_KEY=
```
